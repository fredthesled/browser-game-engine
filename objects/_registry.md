# Objects Registry

Authoritative list of all reusable GameObject types in this repo. When you create a new object type, add a row here in the same session. When you delete or rename one, update this immediately.

## Format

Each entry includes the type name, the file path, current status, a one-line purpose, and the scripts typically attached to instances of this type.
